```java

// ❌ 差：不清晰
```

**方法前缀：**
```java
// ✅ 好：使用标准前缀
public BlockState polymer$getPolymerBlockState(...)

// ❌ 差：自定义前缀可能冲突
```

**字段命名：**
```java
// ✅ 好：使用 @Unique 注解和前缀
@Unique
private BlockState polymer$cachedState;

// ❌ 差：可能与目标类字段冲突
private BlockState cachedState;
```

## 性能优化

### 缓存策略

**场景 1：固定伪装（推荐）**
```java
@Unique
private final BlockState polymer$polymerBlockState;

@Inject(method = "<init>", at = @At("TAIL"))
private void initPolymer(CallbackInfo ci) {
    // 构造时初始化一次
    this.polymer$polymerBlockState = Blocks.STONE.defaultBlockState();
}

public BlockState polymer$getPolymerBlockState(...) {
    // 直接返回缓存
    return polymer$polymerBlockState;
}
```

**场景 2：状态依赖伪装**
```java
@Unique
private final BlockState[] polymer$stateCache = new BlockState[16];

@Inject(method = "<init>", at = @At("TAIL"))
private void initPolymer(CallbackInfo ci) {
    // 预计算所有可能的状态组合
    for (Direction dir : Direction.values()) {
        for (AnvilBlock.Damage damage : AnvilBlock.Damage.values()) {
            int index = calculateIndex(dir, damage);
            polymer$stateCache[index] = createPolymerState(dir, damage);
        }
    }
}

public BlockState polymer$getPolymerBlockState(BlockState state, PacketContext context) {
    int index = calculateIndex(
        state.getValue(FACING),
        state.getValue(DAMAGE)
    );
    return polymer$stateCache[index];
}
```

**场景 3：玩家特定伪装**
```java
@Unique
private final Map<UUID, BlockState> polymer$playerCache = new ConcurrentHashMap<>();

public BlockState polymer$getPolymerBlockState(BlockState state, PacketContext context) {
    ServerPlayer player = context.getPlayer();
    if (player == null) {
        return getDefaultPolymerState(state);
    }
    
    // 为每个玩家缓存
    return polymer$playerCache.computeIfAbsent(
        player.getUUID(),
        uuid -> computePolymerStateForPlayer(state, player)
    );
}
```

### 避免重复计算

**❌ 差：每次都重新计算**
```java
public BlockState polymer$getPolymerBlockState(BlockState state, PacketContext context) {
    BlockState vanilla = Blocks.ANVIL.defaultBlockState();
    
    // 每次都创建新对象
    if (state.hasProperty(FACING)) {
        vanilla = vanilla.setValue(FACING, state.getValue(FACING));
    }
    
    if (state.hasProperty(LEVEL_ANVIL)) {
        int level = state.getValue(LEVEL_ANVIL);
        vanilla = vanilla.setValue(DAMAGE, AnvilBlock.Damage.values()[level]);
    }
    
    return vanilla;
}
```

**✅ 好：预计算 + 查表**
```java
@Unique
private static final Map<BlockState, BlockState> CONVERSION_MAP = new HashMap<>();

static {
    // 启动时预计算所有可能的转换
    for (BlockState state : CustomBlock.POSSIBLE_STATES) {
        BlockState vanilla = Blocks.ANVIL.defaultBlockState();
        // 转换逻辑...
        CONVERSION_MAP.put(state, vanilla);
    }
}

public BlockState polymer$getPolymerBlockState(BlockState state, PacketContext context) {
    // O(1) 查找
    return CONVERSION_MAP.getOrDefault(state, DEFAULT_STATE);
}
```

### 内存管理

**使用弱引用缓存：**
```java
@Unique
private static final Map<BlockState, BlockState> CACHE = 
    Collections.synchronizedMap(new WeakHashMap<>());

// WeakHashMap 会在内存压力大时自动清理
```

**限制缓存大小：**
```java
@Unique
private static final int MAX_CACHE_SIZE = 1000;

@Unique
private static final Map<BlockState, BlockState> CACHE = 
    Collections.synchronizedMap(new LinkedHashMap<>(MAX_CACHE_SIZE, 0.75f, true) {
        @Override
        protected boolean removeEldestEntry(Map.Entry<BlockState, BlockState> eldest) {
            return size() > MAX_CACHE_SIZE;
        }
    });
```

```java
}
```

### 异常处理

**捕获特定异常：**
```java
public BlockState polymer$getPolymerBlockState(BlockState state, PacketContext context) {
    try {
        return computePolymerState(state, context);
    } catch (IllegalArgumentException e) {
        // 记录错误但不崩溃
        LOGGER.error("Failed to compute polymer state for {}: {}", state, e.getMessage());
        return Blocks.STONE.defaultBlockState();
    }
}
```

**优雅降级：**
```java
public BlockState polymer$getPolymerBlockState(BlockState state, PacketContext context) {
    // 尝试完整转换
    BlockState result = tryFullConversion(state, context);
    if (result != null) {
        return result;
    }
    
    // 降级到简单转换
    result = trySimpleConversion(state);
    if (result != null) {
        return result;
    }
    
    // 最后的后备方案
    return Blocks.STONE.defaultBlockState();
}
```
