## JDK 安装与配置

### 下载 JDK

**推荐发行版：**
- Adoptium Eclipse Temurin
- Oracle OpenJDK
- Amazon Corretto

**下载地址：**
```
https://adoptium.net/
```

### 验证安装

```bash
# 检查 Java 版本
java -version
# 输出应包含 "25" 或更高

# 检查 javac
javac -version

# 检查 JAVA_HOME
echo $JAVA_HOME  # Linux/Mac
echo %JAVA_HOME%  # Windows
```

### 配置环境变量（Windows）

```batch
# 设置 JAVA_HOME
setx JAVA_HOME "C:\Program Files\Java\jdk-25"

# 添加到 PATH
setx PATH "%PATH%;%JAVA_HOME%\bin"
```

## Gradle 配置

### 使用项目自带的 Gradle Wrapper

**推荐方式：** 使用项目的 `gradlew` 脚本

```bash
# Linux/Mac
./gradlew build

# Windows
gradlew.bat build
```

**好处：**
- 自动下载正确版本的 Gradle
- 确保构建一致性
- 无需手动安装

### 全局 Gradle 安装（可选）

```bash
# macOS (Homebrew)
brew install gradle

# Linux (SDKMAN)
sdk install gradle

# 验证
gradle --version
```

### Gradle 缓存配置

**加速下载（国内用户）：**

编辑 `~/.gradle/init.gradle`：
```gradle
allprojects {
    repositories {
        maven { url 'https://maven.aliyun.com/repository/public/' }
        maven { url 'https://maven.aliyun.com/repository/google/' }
        mavenCentral()
    }
}
```

## IDE 配置

### IntelliJ IDEA（推荐）

**导入项目：**
1. 打开 IDEA
2. File → Open
3. 选择项目的 `build.gradle` 文件
4. 等待 Gradle 同步完成

**推荐插件：**
- Minecraft Development
- Fabric Language
- JSON Schema Validator

**JDK 配置：**
1. File → Project Structure
2. Project → Project SDK → 选择 JDK 25
3. Modules → Language level → 25

**必需插件：**
- Extension Pack for Java
- Gradle for Java

**配置 settings.json：**
```json
{
  "java.configuration.runtimes": [
    {
      "name": "JavaSE-25",
      "path": "/path/to/jdk-25"
    }
  ]
}
```

## 项目结构理解

### 单模块项目

```
modname/
├── src/
│   └── main/
│       ├── java/           # Java 源代码
│       │   └── com/example/mod/
│       │       ├── ModMain.java
│       │       ├── block/
│       │       └── item/
│       └── resources/      # 资源文件
│           ├── fabric.mod.json
│           ├── modid.mixins.json
│           └── assets/
├── build.gradle           # 构建配置
├── gradle.properties      # 项目属性
└── settings.gradle        # Gradle 设置
```

### 多模块项目（Common + Fabric）

```
modname/
├── Common/               # 平台无关代码
│   ├── src/
│   │   └── main/
│   │       ├── java/
│   │       └── resources/
│   └── build.gradle
├── Fabric/              # Fabric 特定代码
│   ├── src/
│   │   └── main/
│   │       ├── java/
│   │       └── resources/
│   └── build.gradle
├── build.gradle         # 根配置
└── settings.gradle
```

## 常用配置文件

### gradle.properties

```properties
# Minecraft 版本
minecraft_version=26.1.2
yarn_mappings=26.1.2+build.1

# Fabric
loader_version=0.18.4
fabric_version=0.144.3+26.1

# Polymer
polymer_version=0.16.5+26.1.2

# 模组信息
mod_version=1.0.0
maven_group=com.example.mod
archives_base_name=modname
```

### fabric.mod.json

```json
{
  "schemaVersion": 1,
  "id": "modid",
  "version": "${version}",
  "name": "Mod Name",
  "environment": "*",
  "entrypoints": {
    "main": ["com.example.mod.ModMain"]
  },
  "mixins": ["modid.mixins.json"],
  "depends": {
    "fabricloader": ">=0.18.4",
    "fabric-api": "*",
    "minecraft": "26.1.2"
  }
}
```

## 构建系统设置

### 加速构建

**启用 Gradle 守护进程：**

编辑 `gradle.properties`：
```properties
org.gradle.daemon=true
org.gradle.parallel=true
org.gradle.caching=true
```

**分配更多内存：**
```properties
org.gradle.jvmargs=-Xmx4G
```

### 离线模式

```bash
# 首次构建（下载依赖）
./gradlew build

# 后续离线构建
./gradlew build --offline
```

## 测试环境搭建

### 本地服务器测试

**创建测试服务器：**
```bash
# 下载 Fabric 服务器
wget https://meta.fabricmc.net/v2/versions/loader/26.1.2/0.18.4/1.0.1/server/jar

# 重命名
mv server.jar fabric-server.jar

# 首次运行（生成文件）
java -Xmx2G -jar fabric-server.jar nogui

# 同意 EULA
echo "eula=true" > eula.txt

# 复制模组
mkdir mods
cp build/libs/*.jar mods/

# 启动服务器
java -Xmx2G -jar fabric-server.jar nogui
```

### 客户端测试

**安装 Fabric Loader：**
1. 下载 Fabric Installer：https://fabricmc.net/use/installer/
2. 运行安装器
3. 选择 Minecraft 版本 26.1.2
4. 点击 Install

**安装模组：**
```bash
# Windows
cp build/libs/*.jar "%APPDATA%\.minecraft\mods\"

# Mac
cp build/libs/*.jar "~/Library/Application Support/minecraft/mods/"

# Linux
cp build/libs/*.jar "~/.minecraft/mods/"
```

## 常见环境问题

### 问题 1：找不到 JDK

**错误：**
```
ERROR: JAVA_HOME is not set
```

**解决：**
```bash
# 查找 Java 安装路径
which java  # Linux/Mac
where java  # Windows

# 设置 JAVA_HOME
export JAVA_HOME="/path/to/jdk-25"  # Linux/Mac
setx JAVA_HOME "C:\path\to\jdk-25"  # Windows
```

---

### 问题 2：Gradle 版本不兼容

**错误：**
```
Unsupported class file major version 69
```

**解决：**
- Java 25 需要 Gradle 9.2+
- 更新 `gradle/wrapper/gradle-wrapper.properties`：
```properties
distributionUrl=https\://services.gradle.org/distributions/gradle-9.2.1-bin.zip
```

---

### 问题 3：内存不足

**错误：**
```
OutOfMemoryError: Java heap space
```

**解决：**

编辑 `gradle.properties`：
```properties
org.gradle.jvmargs=-Xmx4G -Xms2G
```
